Whats is HTML



Hypertext Markup Language (HTML) is the stadart markup language for creating web pages and web applications

Web browserrs receive HTML documents from a web server of from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the apperance of the documents

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive formsmay be embedded into the rendered page. HTML provides a meants to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes anr other items. HTML elements are delineated by tags, written using angle brackets. Tags such as <img/> and <input/> directly introduce content inte the page. Other tags such as <p> surround and provide information about documents text and may include other tags as sub-elements. Browsers do not display the HTML tags, but use them to interpret the content or the page.


Markup



HTML markup consist of several ley components, including those called tags (and their attributes), character-based types, character references and references. HTML tags most comonnly come in pairs like <h1> and </h1>, although some represent empty elements and so areunpaired, for example <img>. The first tag in such a pair is the start tag, and the second is the end tag(they are also called openning tags and closing tags)

Another important component is the HTML documents type declaration, which truggers standads mode rendering
